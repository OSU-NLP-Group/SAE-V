Module saev.probing

This is a quick and dirty script to run some probing tasks and make some visualizations for a preprint. It is not clean, variables are very poorly named, and I will (hopefully) clean this code up massively in the coming weeks.

That said, here's how you run it:

First, record the activations for the probing images:

uv run main.py activations --dump-to /local/scratch/stevens.994/cache/saev/ --model-org open-clip --model-ckpt ViT-B-16/openai --d-vit 768 --n-layers 2 --n-patches-per-img 196 data:probe-dataset --data.root /research/nfs_su_809/workspace/stevens.994/datasets/saev-probing/

Then run the probe task:

uv run main.py probe --ckpt checkpoints/1lm0m9d2/sae.pt --data.shard-root /local/scratch/stevens.994/cache/saev/91f22b6ab60f337d22a834dc77f6849445eabc5ce51e0875d296e9b39398b2e4/ --data.patches patches --data.layer -2 --images.root /research/nfs_su_809/workspace/stevens.994/datasets/saev-probing --dump-to /research/nfs_su_809/workspace/stevens.994/saev/probing/1lm0m9d2

class FinegrainedTrait (root: str, examples: dict[str, list[str]], *, transform=None)

An abstract class representing a :class:Dataset.

All datasets that represent a map from keys to data samples should subclass it. All subclasses should overwrite :meth:__getitem__, supporting fetching a data sample for a given key. Subclasses could also optionally overwrite :meth:__len__, which is expected to return the size of the dataset by many :class:~torch.utils.data.Sampler implementations and the default options of :class:~torch.utils.data.DataLoader. Subclasses could also optionally implement :meth:__getitems__, for speedup batched samples loading. This method accepts list of indices of samples of batch and returns list of samples.

Note

:class:~torch.utils.data.DataLoader by default constructs an index sampler that yields integral indices. To make it work with a map-style dataset with non-integral indices/keys, a custom sampler must be provided.

class FinegrainedTrait(torch.utils.data.Dataset):
    def __init__(self, root: str, examples: dict[str, list[str]], *, transform=None):
        self.samples = []
        self.label_lookup = {}
        for label, paths in sorted(examples.items()):
            if label not in self.label_lookup:
                self.label_lookup[label] = len(self.label_lookup)

            for path in paths:
                self.samples.append({
                    "path": os.path.join(root, path),
                    "label": label,
                    "target": self.label_lookup[label],
                })

        self.transform = transform

    def __getitem__(self, index: int) -> dict[str, object]:
        """
        Args:
            index: Index

        Returns:
            dict with keys 'image', 'index' and 'target'.
        """
        example = self.samples[index]
        image = Image.open(example.pop("path")).convert("RGB")
        if self.transform is not None:
            image = self.transform(image)
        example["image"] = image
        example["index"] = index
        return example
